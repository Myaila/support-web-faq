# Formas de Pagamento

Aceitamos diferentes meios de pagamento para a sua comodidade.

## Meios disponíveis

| Meio de pagamento | Prazo de confirmação |
|-------------------|----------------------|
| Cartão de crédito | Imediato             |
| Cartão de débito  | Imediato             |
| PIX               | Imediato             |
| Boleto bancário   | Até 3 dias úteis     |

## Como adicionar um cartão

1. Acesse **Configurações** > **Pagamentos**.
2. Clique em **Adicionar cartão**.
3. Preencha os dados do cartão.
4. Clique em **Salvar**.

## Segurança dos dados

Todas as transações são processadas com criptografia SSL e seguem os padrões PCI-DSS.
